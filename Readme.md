# Programação de Computadores I (ISTEC)

Bem-vindo ao meu repositório! Chamo-me Rodrigo e sou estudante de Engenharia Multimédia no ISTEC.

Aqui partilho todos os meus exercícios e trabalhos feitos na unidade curricular **Programação de Computadores I**. Uso este repositório para organizar o meu estudo e mostrar o meu progresso em Python — pode servir também como parte do meu portefólio.

---

## Sobre a cadeira
- **Curso:** Licenciatura em Engenharia Multimédia (ISTEC)
- **Cadeira:** Programação de Computadores I
- **Ano letivo:** 2024/2025
- **Linguagem:** Python

O objetivo da disciplina é dar-nos bases sólidas de programação, usando Python como linguagem introdutória. Aprendemos a resolver problemas, a estruturar programas e a utilizar ferramentas de desenvolvimento.

## Temas principais abordados na disciplina
- Variáveis, tipos de dados e operadores
- Estruturas de decisão (if, else, elif)
- Ciclos (for, while)
- Funções e modularização
- Estruturas de dados: listas, tuplos, dicionários
- Entrada e saída de dados
- Programação orientada a objetos (introdução)
